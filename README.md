# FredPerr's dotfiles

## Getting Started

Clone the repository with the following command:

`git clone https://github.com/FredPerr/.dotfiles/ ~/.dotfiles`

_You may have to install `git` before with `sudo apt-get install git`._ 

### Install

Run the `install/install.sh` file with `sudo`:
`sudo bash ~/.dotfiles/install/install.sh`

You might want to re-run this script if there are errors of failures.

Then, restart your computer.

** That should be it!**
